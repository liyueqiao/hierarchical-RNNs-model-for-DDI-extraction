Note that DDIextraction_2013.tar.gz is the original DDI corpus from DDI extraction 2013 challenge, which contains 714 train files and 191 test files. Since sometrain files in the original corpus donot contain DDI, we remove these train files from DDIextraction_2013 to construct the new DDI corpus named DDI2013_preprocess.tar.gz. There are 624 train files and 191 test files in the DDI2013_preprocess corpus.
